# 👋 Hi, I'm Debangshi

I'm a CS student who decided to get **seriously serious** about data.

For a long time I had the degree but not the skills. I could write code here and there, but I wasn't truly confident — not in SQL, not in statistics, not in the tools that actually matter for a data analyst job.

So I made a decision: **90 days. Zero to interview-ready. No shortcuts.**

This GitHub is my proof of work. Every query I write, every project I build, every concept I finally understand — it lives here. Raw, honest, and real.

---

## 🎯 The Goal

Land a **Data Analyst job (6+ LPA)** as a fresher by being genuinely skilled — not just certificate-collecting.
